# Presentación Interactiva del Trabajo de Fin de Máster

Este proyecto contiene una presentación web interactiva para la defensa del Trabajo de Fin de Máster (TFM) titulado "Detección de Fraudes en Tarjetas de Crédito en el Sector Bancario".

La presentación está desarrollada en HTML, CSS y JavaScript, utilizando la librería Chart.js para la visualización de datos animados.

## Enlaces Relevantes

- **Presentación en Vivo (Firebase):** [https://presentacion-tfm-25599.web.app/](https://presentacion-tfm-25599.web.app/)
- **Repositorio del TFM (Código y Documento):** [https://github.com/ronnysuero/tfm_deteccion_de_fraudes_tarjeta_de_credito](https://github.com/ronnysuero/tfm_deteccion_de_fraudes_tarjeta_de_credito)

## Visualización Local

Para ver la presentación en tu máquina local:

1.  Asegúrate de tener los tres archivos en la misma carpeta:
    - `index.html`
    - `umecit.png`
    - `immune.png`
2.  Abre el archivo `index.html` en cualquier navegador web moderno.
